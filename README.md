# 🩸 Blood Bank Management System

## 📌 Project Overview
The **Blood Bank Management System** is a web-based application developed using **Python (Flask), MySQL, HTML, CSS, and JavaScript**.  
The system manages blood donors, hospitals, blood banks, and blood requests efficiently.

This project provides an interface for administrators to **add, update, delete, and analyze blood donation data**. It also helps users check blood availability and request blood from hospitals.

---

## 🎯 Objectives

- Maintain records of **donors, hospitals, blood banks, and acceptors**
- Manage **blood donations and blood availability**
- Perform **SQL query analysis**
- Allow administrators to **add, update, and delete records**
- Provide a **simple and user-friendly web interface**

---

## 🏗 System Architecture

The system consists of three main layers:

### 1️⃣ Presentation Layer
- HTML
- CSS
- JavaScript
- Login Page
- Dashboard
- Query Interface

### 2️⃣ Application Layer
- Python
- Flask Web Framework
- Handles user requests and system logic

### 3️⃣ Database Layer
- MySQL Database
- Stores donor, hospital, blood bank, and donation data
